# DIP 六套 UI 风格 · 修改方案

**原则**：所有正文、标题、按钮文案等**一字不改**，仅通过配色、圆角、阴影、字体与少量布局变量，让每一套主题更贴近对应品牌官网的视觉语言。

---

## 1. Palantir

**参考**：palantir.com、Blueprint 设计系统（深蓝 cerulean、浅蓝灰背景、企业感）

| 项目 | 原值 | 修改后 | 说明 |
|------|------|--------|------|
| 页面背景 | #f5f6f8 | **#F5F8FB** | 浅蓝灰，与 Blueprint 一致 |
| 主色/强调色 | #1e3a5f | **#2D4A6F** / **#3D5A80** | 深蓝灰，偏 cerulean，更易识别 |
| 次要文字 | #475569 | **#5F6D7C** | 深灰蓝，品牌常用灰 |
| 边框 | #e2e8f0 | **#E2E8F0** 略冷 | 保持冷静感 |
| 圆角 | 6px | **4~6px** | 偏方正，企业工具感 |
| 阴影 | 轻 | **0 1px 2px rgba(0,0,0,0.06)** | 更克制 |
| 字体 | Inter | **Inter**（无 Alliance 时保留） | 官网用 Alliance，用 Inter 近似 |

**排版**：导航与卡片保持简洁线条，按钮不做大圆角，整体偏「数据/企业后台」冷静风。

## 实施方式

- 仅修改 `dip-product.html` 内 `[data-theme="..."]` 的 CSS 变量及可选的主题专属 class（如 Hero 渐变）。
- **不**改任何 HTML 内文字、不删减段落、不合并区块。
- 若某品牌需要更强「排版」差异（如 Apple 更大标题字重、MiniMax 更疏行距），可在对应 `[data-theme="xxx"]` 下增加少量覆盖规则（如 `--font-weight-title: 700`、`--line-height-body: 1.6`），再在样式中引用这些变量。

按上述方案调整后，六套风格会明显更贴近各自官网，同时保证文案完整不变。
